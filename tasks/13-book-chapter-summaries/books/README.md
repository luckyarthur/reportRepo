# Place book files here

Drop the book(s) to be summarized into this folder. Supported formats:
`.epub`, `.pdf`, `.txt`, `.md`, `.docx`, `.mobi`.

The task (see `../task.md`) reads every book in this folder and writes a
chapter-by-chapter summary with core knowledge to `../output.md`.

Note: files uploaded only into a chat session are temporary. To have the
scheduled routine process a book, commit the book file into this folder.
